# HR Analytics SQL Project

## Project Overview
This project analyzes employee salary, attendance, performance, and leave data using SQL.

## Tools Used
- MySQL
- MySQL Workbench

## SQL Concepts Used
- JOINS
- GROUP BY
- Aggregations
- Subqueries
- ORDER BY
- LIMIT

## Tables Used
- employees
- departments
- salaries
- attendance
- performance
- leaves

## Key Business Questions
- Which department has highest average salary?
- Which employees are top performers?
- Which department has more leave records?
- Which employees earn above average salary?
- Department-wise employee count analysis

## Key Insights
- Salary trends across departments
- Employee performance tracking
- Leave analysis by department
- Attendance monitoring
